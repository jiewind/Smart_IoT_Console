基于 STM32 + ESP8266 + Python 的非阻塞双向通信物联网控制终端

一、	项目简介

本项目是一个软硬结合的全栈式物联网实战项目。下位机设备最终成品可显示当前温湿度，并根据当前环境灯光调整LED灯亮度。同时，项目建立了全栈控制台，可以在上位机显示温度、湿度、光照强度、以及历史温度趋势。用户还可通过点击按钮控制测试灯的亮灭，切换屏幕显示界面，实现了完整的双向监控闭环。

二、	硬件拓补

本项目使用硬件设备如下：

STM32F103C8T6主控板

DHT11温湿度检测模块

光敏感应器

四针OLED显示屏

ESP8266模块

独立按键模块

注：具体的接线方式请参考工程配套的硬件连接图

三、	软件核心架构

下位机端：底层纯 C 语言非阻塞异步有限状态机驱动，支持断网自动重连与透传逃逸，保障高并发下的系统稳定性。

上位机端：上位机基于 Python 多线程架构开发，内置心跳超时哨兵与本地文件持久化机制，确保数据不丢失且设备状态实时同步。



四、	核心目录结构

.vscode/：里面包含的文件是本项目在vscode运行设置的运行及烧录键，实现一键编译下载。

&#x09;Hardware/：里面是硬件的驱动层，主要负责写硬件的各种功能，与业务逻辑完全解耦。

&#x09;App/：里面是应用层内容，主要负责使用各种功能，调度底层驱动。

五、	开发环境与特别说明

开发工具链：本项目程序使用VScode编写，用CubeMX生成配置文件，采用Makefile架构。

运行环境说明：另外本项目在linux系统上实现，不同系统可能程序运行有差别。

研发理念：并且研究过程中使用AI协助。研究本项目旨在加强自己与AI协作能力，并不指望完全学会所有内容，而是探索现代化的开发范式。
